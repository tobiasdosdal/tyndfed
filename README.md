# Tyndfed

Portfolio website built with TanStack Start.

## Development

```bash
bun install
bun dev
```

## Build

```bash
bun run build
```

## Deploy

### Vercel

Just push to GitHub and import - Vercel auto-detects TanStack Start.

### Self-host

```bash
bun run build
bun run start
```

## Stack

- TanStack Start
- React 19
- TypeScript
- Vite 7
